# /iterate

자동 반복 개선을 수행합니다.

## 사용법

```
/iterate                   # 현재 작업 반복 개선
/iterate --until 95        # 95% 달성까지
/iterate --max 3           # 최대 3회
```

## 동작

1. 현재 상태 평가 (gap-detector)
2. 개선 필요 여부 판단
3. 개선 사항 자동 적용
4. 재평가
5. 목표 달성까지 반복

## pm-agent 역할

```
pm-agent (메타 에이전트)
    │
    ├── 복잡한 작업 감지
    │
    ├── 적절한 서브에이전트 호출
    │     ├── gap-detector
    │     ├── pdca-iterator
    │     └── 기타 오케스트레이터
    │
    └── 반복 제어
          ├── 진행 상황 모니터링
          ├── 중단 조건 확인
          └── 사용자 개입 최소화
```

## 중단 조건

| 조건 | 조치 |
|------|------|
| 목표 달성 | 완료 보고 |
| 최대 반복 도달 | 사용자에게 보고 |
| 진전 없음 | 다른 접근 제안 |
| 에러 발생 | 사용자 개입 요청 |

## 옵션

| 플래그 | 효과 |
|--------|------|
| `--until` | 목표 일치율 |
| `--max` | 최대 반복 횟수 |
| `--verbose` | 상세 진행 상황 |

## 연관 커맨드

- `/pdca` - 전체 PDCA 사이클
- `/gap` - 갭 분석만
