# /help

사용 가능한 커맨드 목록과 도움말을 표시합니다.

## 사용법

```
/help                      # 전체 커맨드 목록
/help review               # 특정 커맨드 상세
/help --modes              # 행동 모드 설명
/help --agents             # 에이전트 목록
```
